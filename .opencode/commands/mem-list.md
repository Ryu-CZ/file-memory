Direct memory listing - shows all memories.

**Usage**:
```
/mem-list
/mem-list --tag <tag>
```

This is a direct/manual command - no intelligent ranking or summarization. Use `/memory-recall` for intelligent search with relevant summaries.

Examples:
```
/mem-list
/mem-list --tag work
```
