Direct memory search - raw text search across all memories.

**Usage**:
```
/mem-search <query>
```

This is a direct/manual command - returns raw search results. Use `/memory-recall` for intelligent search with ranked results and summaries.

Examples:
```
/mem-search "pytest"
/mem-search auth
```
