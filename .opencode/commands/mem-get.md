Direct memory retrieval by exact key.

**Usage**:
```
/mem-get <key>
```

This is a direct/manual command - no intelligent search or reasoning. Use `/memory-recall` for intelligent search with ranked results.

Examples:
```
/mem-get project_status
/mem-get user_preference_linter
```
